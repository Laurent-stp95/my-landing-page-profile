# My Landing Page PM

Landing page personnelle — Profil Product Manager.

## Stack

- HTML / CSS / JS — pas de framework, pas de build
- Fichiers modulaires (1 fichier CSS par section)
- Responsive mobile

## Structure

```
my-landing-page-pm/
├── index.html
├── css/          ← 1 fichier par section
├── js/           ← navigation, animations, parallax
└── assets/       ← images, favicon
```

## Lancer en local

Ouvrir `index.html` dans un navigateur. C'est tout.

## Déployer sur GitHub Pages

1. Aller dans **Settings** → **Pages**
2. Source : **Deploy from a branch**
3. Branch : `main` / `/ (root)`
4. Cliquer **Save**
5. Le site sera dispo sur `https://TON-USERNAME.github.io/my-landing-page-pm/`
